## 数据备份

### 1. 数据导出
```
# 数据库
mysqldump -u username -p password database_name > backup_file.sql

# 数据表
mysqldump -u username -p password database_name table1 table2 > tables_backup.sql

# 表结构 --no-data
mysqldump -u username -p password --no-data database_name table_name > structure_backup.sql

# 压缩
mysqldump -u username -p password database_name | gzip > backup_file.sql.gz
```

### 2. 数据导入
```
# 数据库
mysql -u username -p password database_name < backup_file.sql

# 数据表
mysql -u username -p password database_name table1 table2 < backup_file.sql
```

### datagrip操作
```
# 导出：
1. datagrip导入/导出.sql
2. 右键数据库,选择导入/导出
3. 选择用'mysqldump'导出 or 用'mysql'还原
4. .exe路径在"C:\Program Files\MySQL\MySQL Server 8.0\bin"
5. 导入/导出路径自主选择

# 导入：
1. 命令行导入/导出
2. C:\Program Files\MySQL\MySQL Server 8.0\bin目录下使用cmd
3. mysqldump -u root -p123456 excel > "D:\\dump.sql"
4. 注意：非mysql环境下不需要使用";"
5. error: mysqldump: Couldn't find table: ";"
6. 解决：删掉";"
```
