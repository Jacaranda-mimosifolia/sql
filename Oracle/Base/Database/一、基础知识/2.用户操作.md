# 用户操作

## 一、权限类别

### 1.PDB权限
```
1.系统权限：
CREATE SESSION              允许用户登录到数据库
CREATE TABLE                允许用户创建表
CREATE VIEW                 允许用户创建视图
CREATE PROCEDURE            允许用户创建存储过程和函数

2.对象权限：
SELECT                      允许用户查询特定表或视图
INSERT                      允许用户向特定表插入数据
UPDATE                      允许用户更新特定表的数据
DELETE                      允许用户删除特定表的数据
EXECUTE                     允许用户执行特定的存储过程或函数

3.角色：
PDB_DBA                     管理权限，PDB最高权限
PDB_APP_ADMIN               应用程序，限制性管理权限
PDB_READ_ONLY               只读权限
```

## 二、权限操作

### 1.查看权限
```
SELECT * FROM dba_sys_privs WHERE grantee = 'user_name';         查看系统权限
SELECT * FROM dba_tab_privs WHERE grantee = 'user_name';         查看对象权限
SELECT * FROM dba_role_privs WHERE grantee = 'user_name';        查看角色权限
SELECT * FROM user_sys_privs;                                    查看当前用户权限
```

### 2.赋予权限
```
GRANT privilege TO username;
```

### 3.移除权限
```
REVOKE privilege FROM username;
```

## 三、用户操作

### 1.查看用户
```
SELECT username FROM dba_users;
```

### 2.创建用户
```
CREATE USER username IDENTIFIED BY passwd;
```

### 3.删除用户
```
DROP USER username CASCADE;
```

## 四、实例操作

### 1.创建 PDB 用户
```
# 普通用户，只能用于当前PDB，不能跨越到别的PDB使用
CREATE USER chen IDENTIFIED BY 123456;
GRANT DBA TO chen;
```

### 2.创建 C## 用户
```
# C##是全局用户，通用于所有的PDB
CREATE USER C##chen IDENTIFIED BY 123456;
GRANT DBA TO C##chen;
```
