## DDL- 表操作

### 1. 表操作
```
# 创建
create table 表名(
    字段1 字段类型[comment 注释],
    字段2 字段类型[comment 注释],
    ...
    字段n 字段类型[comment 注释]
)[comment 表注释];

# 查询
show tables;  						# 展示所有
desc 表名;  							# 查看表结构
show create table 表名;  			# 查看指定表的建表语句

# 删除
drop table 表名;  				    # 删除表
truncate table 表名;					# 删除数据

# 修改
alter tabel 旧表名 rename to 新表名;
```

### 2. 数据类型
```
# 整形
TINYINT
SMALLINT
MEDIUMINT
INT
BIGINT

# 浮点型
FLOAT
DOUBLE(长度，小数位数)
DECIMAL

# 字符串
CHAR(n)  			# 储存恒定(n),性能好
VARCHAR(n)  		# 储存实际长度(n+1)

# 文本类型
TINYTEXT
TEXT
MEDIUMTEXT
LONGTEXT

# 日期时间
YEAR				# 年
DATE				# 年月日
TIME				# 时分秒
DATETIME			# 年月日时分秒
TIMESTAMP

# 二进制类型
TINYBLOB
BLOB
MEDIUMBLOB
```

### 3. 其他操作
```
# 数据表拷贝（结构）
create table table1 as select * from table2 where 1=2;
create table table1 like table2;

# 数据表拷贝
create table table1 as select * from table2;

# 插入数据
insert into table1 as select * from table2;

# 数据表拷贝（结构）
select * into table1 from table2;

# 数据表拷贝
select * into new_talble from old_table where 1=2;

# decimal(p,d) 精确值
amount DECIMAL(6,2);
amount列最多可以存储6位数字，小数位数为2位; 因此，amount列的范围是从-9999.99到9999.99
```
