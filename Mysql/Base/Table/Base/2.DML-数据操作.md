## DML - 数据操作
```
DML（Data Manipulation Language）数据操纵语言
对表里的数据做增删改。
常见语句：
INSERT 插入
UPDATE 更新
DELETE 删除
```

### 1. 数据操作
```
# 插入
insert into 表名(字段1,字段2...) values (值1,值2...);		# 指定字段
insert into 表名 values (值1,值2...);					# 全部字段

insert into 表名(字段1,字段2...) values (值1,值2...)[,(值1,值2...)...;
insert into 表名 values (值1,值2...),(值1,值2...)...;		# 插入多条

# 修改
update 表名 set 字段1=值1,字段2=值2...[where 条件];	        # where为指定修改，不写则修改整张表

# 删除
delete from 表名 [where 条件];  # 不写where则删除整张表的数据
```
