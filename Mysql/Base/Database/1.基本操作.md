## DDL-数据库操作

### 1. 数据库操作
```
# 查询
show databases;						            # 展示所有
select database();  				            # 查看当前
show create database database_name;  			# 查看指定库的建库语句

# 创建
create database [if not exists] database_name [default charset character_set] [collate sort];

# 删除
drop database [if exists] database_name;
```
