# 数据表操作

## 一、数据类型

```
字符类型：
char(n)
varchar2(n)

数字类型：
number(n)    -- 22   
number(n,m)  -- n整数+小数，m小数

日期类型：
date

二进制型：
clob        -- 字符 4000
blob        -- 图片、音乐

-- 自增 GENERATED ALWAYS AS IDENTITY
type_id NUMBER(2) GENERATED ALWAYS AS IDENTITY PRIMARY KEY
-- type_id字段1-99自增

-- 级联删除 ON DELETE CASCADE
-- 避免引用不一致的问题
CONSTRAINT fk_tag_type FOREIGN KEY (type_id) REFERENCES TYPE(type_id) ON DELETE CASCADE
-- 如果 TYPE 表中的某个类型 ID 被删除，
-- 那么所有引用该类型 ID 的 TAG 表记录也会被自动删除。

```
